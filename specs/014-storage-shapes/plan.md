# Implementation Plan: Dual storage shapes — GORM + ent with tenant isolation

**Branch**: `014-storage-shapes` | **Date**: 2026-06-11 | **Spec**: `spec.md`

## Summary

Two parallel tracks: (1) add tenant scoping + `LookupByHash` to the existing
GORM shape (`protoc-gen-storage`), and (2) deliver a new ent shape
(`protoc-gen-ent` + `persistence/entrepo/`). Both implement
`persistence.Repository[T,K]`; both enforce tenant isolation at the query layer.
`testdata/apikey` exercises both shapes and the isolation security test passes.

## Technical Context

**New dep**: `entgo.io/ent` added to root `go.mod` (for `persistence/entrepo`)
**Testing**: SQLite in-memory for ent integration tests (no Postgres needed)
**Existing unchanged**: `MemoryRepository`, DSN, `protoc-gen-svc`, runtime, secret

## Architecture

### GORM: tenant injection pattern

```go
// Generated in List, Get, Update, Delete when account_id field present:
tenantID := middleware.TenantIDFromContext(ctx)
if tenantID != "" {
    q = q.Where("account_id = ?", tenantID)
}
```

Simple, explicit, readable in the generated code. Skipped when tenant is empty
(dev mode). Applied in all four mutating/reading methods.

### GORM: LookupByHash

```go
// Generated for each secret:true field named key_value:
func (r *APIKeyRepository) LookupByKeyValueHash(ctx context.Context, hash string) (*APIKey, error) {
    if hash == "" {
        return nil, persistence.ErrNotFound
    }
    tenantID := middleware.TenantIDFromContext(ctx)
    q := r.db.WithContext(ctx).Where("key_value_hash = ?", hash)
    if tenantID != "" {
        q = q.Where("account_id = ?", tenantID)
    }
    var m APIKeyModel
    if err := q.First(&m).Error; err != nil {
        if err == gorm.ErrRecordNotFound { return nil, persistence.ErrNotFound }
        return nil, err
    }
    return fromModel_APIKey(&m), nil
}
```

### ent: generated schema structure

```
ent/
├── schema/
│   ├── mixin.go         NEW (framework-provided) — TenantMixin
│   └── apikey.go        GENERATED by protoc-gen-ent
├── generate.go          GENERATED — //go:generate entc generate ./schema
└── (ent-generated client after go generate ./ent/...)
```

`schema/apikey.go` (generated by protoc-gen-ent):
```go
type APIKey struct{ ent.Schema }

func (APIKey) Mixin() []ent.Mixin {
    return []ent.Mixin{TenantMixin{}} // adds account_id + interceptor
}

func (APIKey) Fields() []ent.Field {
    return []ent.Field{
        field.String("id").StorageKey("id").Immutable(),
        field.String("name"),
        field.String("key_prefix").Optional(),
        // secret field key_value → hash+cipher columns:
        field.String("key_value_hash").Optional().Annotations(
            entsql.Annotation{Column: "key_value_hash"},
        ),
        field.String("key_value_cipher").Optional(),
    }
}

func (APIKey) Indexes() []ent.Index {
    return []ent.Index{
        index.Fields("account_id"),       // from TenantMixin
        index.Fields("key_value_hash"),   // for ValidateAPIKey lookup
    }
}
```

`schema/mixin.go` (framework-provided, NOT generated):
```go
type TenantMixin struct{ mixin.Schema }

func (TenantMixin) Fields() []ent.Field {
    return []ent.Field{
        field.String("account_id").NotEmpty().Immutable(),
    }
}

func (TenantMixin) Interceptors() []ent.Interceptor {
    return []ent.Interceptor{
        ent.InterceptFunc(func(next ent.Querier) ent.Querier {
            return ent.QuerierFunc(func(ctx context.Context, q ent.Query) (ent.Value, error) {
                tenantID := middleware.TenantIDFromContext(ctx)
                if tenantID != "" {
                    // type-specific predicate applied via reflection on the query
                    setTenantFilter(q, tenantID)
                }
                return next.Query(ctx, q)
            })
        }),
    }
}
```

`setTenantFilter` uses a `TenantFilterer` interface that the generated ent
query types implement:
```go
type TenantFilterer interface {
    WhereAccountID(string)
}
// protoc-gen-ent generates this method on the query type
```

### `persistence/entrepo.EntRepository[T,K]`

```go
type EntRepository[T any, K comparable] struct {
    creator  func(ctx, T) (T, error)      // wraps ent client Create
    getter   func(ctx, K) (T, error)      // wraps ent client Get
    lister   func(ctx, persistence.ListOptions) ([]T, string, error)
    updater  func(ctx, K, T, ...string) (T, error)
    deleter  func(ctx, K) error
    enc      secret.Encryptor
}
```

This is a function-based adapter — no reflection, no generics gymnastics.
The generated `NewAPIKeyEntRepository(client *ent.Client, enc secret.Encryptor)`
function wires the ent client methods into the adapter.

### Tradeoffs

| Decision | Chosen | Reason |
|----------|--------|--------|
| ent two-step codegen | `protoc-gen-ent` emits schema; `go generate` runs entc | Standard ent workflow; no entc binary dep in protoc-gen-ent |
| GORM tenant: skip when empty | Yes | Dev/test path uses MemoryRepository or a no-tenant context |
| ent tenant: use interceptor not privacy.Policy | Interceptor | Privacy.Policy requires ent enterprise license for some filter types; interceptor is open-source and sufficient |
| `EntRepository` adapter | Function-based | Avoids deep generics; each service generates its own wiring function |
